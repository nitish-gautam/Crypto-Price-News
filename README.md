# Crypto-Price-News
The intention here is to create a WebApp using RESTful API to fetch Crypto currency prices and the latest Block chain news from Crypto world. 

# SKILL SET REQUIRED:
 Django, Python, BootStrap, HTML, CSS, RESTful API, JSON
